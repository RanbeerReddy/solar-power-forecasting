# Release Notes

This initial release packages the solar power forecasting project as a polished, research-oriented repository.

### Highlights

- Added a professional README with project overview, workflow, and reproducibility guidance.
- Added dependency management and environment setup files.
- Added licensing, citation, contribution, and changelog documentation.
- Added a dedicated data guide and a figures directory for notebook exports.
- Preserved the original research methodology while improving repository presentation.
